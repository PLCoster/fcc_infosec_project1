Stock Price Checker

GET /api/stock-prices

Compare Two Stocks

GET /api/stock-prices

Result:

Submit a form to see result!

Example Usage:

To Get Price and Likes for a Stock:
  • Submit the 'Get Single Stock Price' form above with a valid stock symbol (e.g. GOOG), and optionally select whether to like the stock or not (only a single like is accepted for each stock per IP address).
  • Alternatively, send a GET request to /api/stock-prices with url encoded fields of stock (stock symbol of desired stock, e.g. MSFT), and optionally like=true to like the stock.

    • For Example:
    • Get Google stock price info: /api/stock-prices?stock=GOOG
    • Get MSFT stock price info and like the stock: /api/stock-prices?stock=MSFT&like=true
To Compare the Price and Likes of Two Stocks:
  • Submit the 'Compare Two Stocks' form above with two valid stock symbols (e.g. GOOG / AMZN), and optionally select whether to like both stocks or not (only a single like is accepted for each stock per IP address).
  • Alternatively, send a GET request to /api/stock-prices with two url encoded fields of stock (stock symbol of desired stock, e.g. MSFT), and optionally like=true to like both stocks.

    • For Example:
    • Get Google and Microsoft stock price and like info: /api/stock-prices?stock=GOOG&stock=AMZN
    • Get MSFT and Amazon stock price info, and like both stocks: /api/stock-prices?stock=MSFT&stock=AMZN&like=true

Example Output:

On requesting price information for a single stock:

{
    "stockData": {
    • "stock": "GOOG",
    • "price": "103.21",
    • "likes": 23,
    }
}

On requesting price information for two stocks:

{
    "stockData": [
    {
    • "stock": "GOOG",
    • "price": "103.21",
    • "likes": 23,
    • "rel_likes": -4,
    },
    {
    • "stock": "MSFT",
    • "price": "242.59",
    • "likes": 27,
    • "rel_likes": 4,
    }]

If an invalid stock symbol is sent, or a required stock query parameter is missing, a JSON with an 'error' property is returned explaining the problem.
